# Data Drift Detector

## Learning Objectives
- Understand data drift detection concepts
- Learn PSI-like drift metrics
- Practice monitoring input distributions

## How to Run
1. Open `index.html` in a browser
2. Click "Start Drift Simulation" to see drift changes
3. Click "Reset" to stop and reset

## Features
- Reference vs current distribution comparison
- PSI-like drift score calculation
- Visual histogram comparison
- Color-coded alerts (red when drift is high)
