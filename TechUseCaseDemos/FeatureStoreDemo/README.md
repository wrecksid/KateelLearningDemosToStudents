# Feature Store Demo

Interactive demo for understanding feature stores in ML systems.

## Learning Objectives

- Understand feature lifecycle management
- Learn about feature versioning
- Explore feature discovery and reuse
- Understand offline-online consistency

## How to Run

1. Open `index.html` in browser
2. Browse feature catalog
3. View feature details
4. Simulate feature serving

## Key Concepts Demonstrated

- **Feature Registry**: Central catalog of features
- **Feature Versioning**: Tracking changes over time
- **Feature Lineage**: Source → Transformation → Serving
- **Offline/Online Consistency**: Same features in training and serving
- **Feature Reuse**: Avoiding duplicate engineering
