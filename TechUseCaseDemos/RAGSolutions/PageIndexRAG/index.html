📖 PageIndex RAG Approach

  1. Index: Create inverted index of document pages
  2. Retrieve: Find pages matching query keywords
  3. Cite: Return page numbers with results
  4. Generate: LLM answers with precise citations

Advantage: Enables exact source attribution for factual accuracy.
