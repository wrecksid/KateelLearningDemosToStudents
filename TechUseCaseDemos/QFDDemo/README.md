# Quality Function Deployment (QFD) Demo

Interactive demo for prioritizing AI features using Quality Function Deployment methodology.

## Learning Objectives

- Understand QFD matrix structure
- Learn customer requirement translation
- Explore feature prioritization techniques
- Practice AI product planning

## How to Run

1. Open `index.html` in browser
2. Enter customer requirements
3. See animated QFD matrix
4. Review prioritized feature list

## Key Concepts Demonstrated

- **Customer Requirements (CRs)**: What customers need
- **Technical Requirements (TRs)**: How to deliver
- **Correlation Matrix**: CR-TR relationships
- **Prioritization**: Feature ranking based on importance
- **AI Enablement**: Mapping AI capabilities to requirements
