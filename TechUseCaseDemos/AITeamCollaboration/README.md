# AI Team Collaboration

## Learning Objectives
- Understand RACI roles in AI projects
- Assign responsibilities to team members
- Document accountability for AI outcomes

## How to Run
1. Open `index.html` in a browser
2. Add team members with roles and responsibilities
3. View team matrix

## RACI Roles
- **Responsible**: Who does the work
- **Accountable**: Who is answerable for outcomes
- **Consulted**: Who provides input
- **Informed**: Who needs to know
