# AI Governance Scorecard

## Learning Objectives
- Assess AI ethics and compliance readiness
- Identify governance gaps in AI projects
- Build checklist-driven compliance workflows

## How to Run
1. Open `index.html` in a browser
2. Check off completed governance items
3. View compliance score and status

## Checklist Items
- Fairness and Bias Mitigation
- Model Transparency/Explainability
- Privacy Protection (GDPR/CCPA)
- Safety and Security Measures
- Audit Trail and Documentation
- Model Validation and Testing
- Ongoing Monitoring Plan
- Governance Committee
