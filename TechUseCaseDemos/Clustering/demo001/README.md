# Clustering Demo 001 — Bank Customer Segmentation

**Repository:** [KateelLearningDemosToStudents](https://github.com/VinayaSharada/KateelLearningDemosToStudents)
**Author:** Professor Vinaya Sathyanarayana
**Section:** `TechUseCaseDemos/Clustering/`

Demonstrates four clustering algorithms on synthetic banking customer data —
building intuition for when each method works best before applying them to
real-world segmentation problems.

---

## What This Demo Does

Loads `syntheticdata.csv` (generated by `syntheticdata.py`) and applies:

| Algorithm | When It Works Best |
|-----------|--------------------|
| K-Means | Spherical, well-separated clusters of similar size |
| DBSCAN | Irregular shapes, noise/outlier handling |
| Gaussian Mixture Models (GMM) | Overlapping clusters with probabilistic membership |
| PCA + K-Means | High-dimensional data needing dimensionality reduction first |

Evaluates each with: **Silhouette Score**, **Calinski-Harabasz Index**, **Davies-Bouldin Index**.

---

## Files

| File | Purpose |
|------|---------|
| `clusteringdemo.py` | Main demo — all four algorithms with visualisations |
| `syntheticdata.py` | Generates `syntheticdata.csv` with numerical, categorical, and date features |
| `requirements.txt` | Python dependencies |

---

## Setup & Run

```bash
cd TechUseCaseDemos/Clustering/demo001
pip install -r requirements.txt
python syntheticdata.py      # generate data first
python clusteringdemo.py
```

---

## Expected Output

Printed cluster metrics for each algorithm, plus matplotlib plots showing:
- Cluster distributions
- Silhouette analysis
- PCA 2D projection coloured by cluster assignment

## Student Extensions

1. Vary `k` from 2 to 8 and plot the **elbow curve** to find the optimal number of segments.
2. Add a `RapidMiner` export step and compare results with the `CustSeg/rmp_scripts/` templates.
3. Assign business names to each cluster (e.g. "High-Value Savers", "Dormant Accounts").
4. Replace GMM with **Agglomerative Clustering** and compare the dendrograms.
