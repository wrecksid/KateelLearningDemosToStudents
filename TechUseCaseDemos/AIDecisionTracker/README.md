# AI Decision Tracker

## Learning Objectives
- Document AI product decisions systematically
- Build decision rationale through examples
- Create an audit trail for AI development

## How to Run
1. Open `index.html` in a browser
2. Fill out the decision form
3. View decision log (persisted in localStorage)

## Features
- Category tagging (Model, Data, Deployment, Ethics)
- Date-stamped decisions
- Persistent storage
