# AI ROI Calculator

Calculator for estimating the return on investment for AI projects.

## Learning Objectives

- Understand AI project economics
- Learn cost-benefit analysis
- Explore value realization timelines
- Understand TCO of AI implementations

## How to Run

1. Open `index.html` in browser
2. Enter project parameters
3. See ROI analysis
4. Compare scenarios

## Key Concepts Demonstrated

- **Total Cost of Ownership**: Development + Maintenance
- **Value Drivers**: Efficiency, accuracy, automation
- **Payback Period**: Time to recover investment
- **NPV**: Net Present Value
- **Business Impact**: Quantifiable benefits
