# AI Resource Planner

## Learning Objectives
- Estimate compute resources for AI workloads
- Understand cost implications of model choices
- Learn scaling principles for inference

## How to Run
1. Open `index.html` in a browser
2. Select model type and enter usage parameters
3. View estimated resources and costs

## Resource Factors
- **Model Type**: Determines baseline requirements
- **Inference Volume**: Scales compute linearly
- **Latency Target**: Lower latency = more resources
- **Duration**: Affects total cost calculation
