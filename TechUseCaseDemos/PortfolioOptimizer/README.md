# Portfolio Optimizer

Interactive portfolio optimization demo using mean-variance optimization (Markowitz model).

## Learning Objectives

- Understand Modern Portfolio Theory (MPT)
- Learn about efficient frontier
- Explore risk-return tradeoff
- Understand diversification benefits

## How to Run

1. Open `index.html` in browser
2. Adjust portfolio weights using sliders
3. See portfolio return, risk, and Sharpe ratio
4. Compare with efficient frontier

## Key Concepts Demonstrated

- **Mean-Variance Optimization**: Risk-return tradeoff
- **Efficient Frontier**: Optimal portfolios
- **Sharpe Ratio**: Risk-adjusted returns
- **Correlation**: Diversification effects
- **Asset Allocation**: Weight optimization
