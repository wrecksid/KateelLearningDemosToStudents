# Credit Scoring Dashboard

Interactive demo for understanding credit scoring models and feature engineering.

## Learning Objectives

- Understand logistic regression for credit scoring
- Explore feature importance and scorecard visualization
- Learn about fair lending and bias detection

## How to Run

1. Open `index.html` in browser
2. Adjust applicant profile using sliders
3. See real-time credit score and decision
4. Explore the feature contribution chart

## Key Concepts Demonstrated

- **Logistic Regression**: Probability-based scoring
- **Feature Engineering**: Income, debt ratio, history length
- **Scorecard**: Points-based system visualization
- **Fairness**: Equal opportunity across demographics
- **Business Rules**: Minimum score thresholds
