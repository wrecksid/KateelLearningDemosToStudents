# A/B Testing Framework

Interactive demo for A/B testing with statistical significance.

## Learning Objectives

- Understand hypothesis testing
- Learn p-value and confidence intervals
- Explore sample size calculations
- Understand business impact of testing

## How to Run

1. Open `index.html` in browser
2. Configure test parameters
3. Generate simulated results
4. See statistical analysis

## Key Concepts Demonstrated

- **Null Hypothesis**: No difference between variants
- **P-value**: Probability of observing results
- **Confidence Level**: 95% standard
- **Sample Size**: Power analysis
- **Statistical Significance**: Decision criteria
