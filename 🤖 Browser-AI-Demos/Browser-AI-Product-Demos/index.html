These demos cannot run when opened as a file:// URL.
Chrome blocks ES module imports and HuggingFace CDN fetches from the file:// protocol.

Fix — choose one option:
  1. Double-click start_demos.bat in this folder (Windows) — launches Python server & opens browser automatically.
  2. Open a terminal in this folder and run: python -m http.server 8080 then visit http://localhost:8080
  3. Use VS Code Live Server extension — right-click index.html → "Open with Live Server".
