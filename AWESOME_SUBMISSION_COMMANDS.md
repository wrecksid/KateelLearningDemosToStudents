# Awesome List Submission Commands

The following commands can be run to submit to awesome lists. First, fork each repository, then run:

## 1. awesome-machine-learning (josephmisiti/awesome-machine-learning)

```bash
# After forking
cd awesome-machine-learning
git checkout -b add-kateel-demos
# Edit README.md in Education section
git add README.md
git commit -m "Add KateelLearningDemosToStudents"
git push origin add-kateel-demos
gh pr create --repo josephmisiti/awesome-machine-learning --title "Add KateelLearningDemosToStudents to Education" --body "Free AI/ML demos for students - 33 demos, zero cloud, synthetic data, browser-based RAG solutions."
```
