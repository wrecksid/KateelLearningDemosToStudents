# Aircraft IoT Intrusion Detection System

## Learning Objectives
- Understand intrusion detection concepts
- Learn aircraft IoT network monitoring
- Practice response to security alerts

## How to Run
1. Open `index.html` in a browser
2. Monitor real-time connection status
3. Review security alerts

## Features
- Real-time network connection monitoring
- Anomaly detection for aircraft protocols
- Alert prioritization by severity

## Aircraft Protocols Monitored
- **CAN Bus**: Controller Area Network for vehicle systems
- **ARINC**: Aviation data buses
- **TCP/UDP**: Standard network protocols
