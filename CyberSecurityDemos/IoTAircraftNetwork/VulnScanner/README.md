# Aircraft IoT Vulnerability Scanner

## Learning Objectives
- Identify common IoT vulnerabilities in aviation
- Learn vulnerability assessment techniques
- Understand aircraft-specific attack vectors

## How to Run
1. Open `index.html` in a browser
2. Select a target device
3. Click "Start Vulnerability Scan"
4. Review results by severity

## Common Aircraft IoT Vulnerabilities
- **Critical**: Buffer overflows, command injection
- **High**: Authentication bypasses, encryption weaknesses
- **Medium**: Data exposure, lack of integrity checks
- **Low**: Information disclosure, weak access controls
