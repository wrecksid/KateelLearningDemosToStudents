# Aircraft IoT Network Traffic Analyzer

## Learning Objectives
- Analyze aircraft IoT network protocols
- Identify traffic patterns
- Detect anomalies in real-time

## How to Run
1. Open `index.html` in a browser
2. Monitor live traffic statistics
3. View protocol distribution chart

## Aircraft Protocols
- **CAN Bus**: Engine, flight control systems
- **ARINC 429**: Avionics data bus
- **TCP**: Ground communications
- **UDP**: Sensor data streaming

## Key Metrics
- Packet rates per protocol
- Anomaly detection count
- Real-time traffic log
