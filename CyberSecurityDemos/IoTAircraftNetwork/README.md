# Aircraft IoT Cyber Security Demos

## Overview
This directory contains browser-based cyber security demos for aircraft IoT systems.

## Demos

### 1. Penetration Testing Simulator
- **Path:** `PenTestSimulator/`
- **Description:** Simulate penetration testing on aircraft IoT systems
- **Features:** Port scanning, service detection, exploit simulation, patch management

### 2. Threat Modeling Matrix
- **Path:** `ThreatModelingMatrix/`
- **Description:** STRIDE threat analysis for aircraft IoT systems
- **Features:** Threat categorization, asset mapping, risk scoring

### 3. Intrusion Detection System
- **Path:** `IntrusionDetection/`
- **Description:** Real-time network monitoring and anomaly detection
- **Features:** Connection monitoring, alert generation, status visualization

### 4. Network Traffic Analyzer
- **Path:** `NetworkTrafficAnalyzer/`
- **Description:** Protocol analysis and anomaly detection
- **Features:** CAN bus, ARINC, TCP/UDP monitoring, traffic statistics

### 5. Vulnerability Scanner
- **Path:** `VulnScanner/`
- **Description:** Automated security assessment for IoT components
- **Features:** CVE database, severity classification, device targeting

## Common Aircraft IoT Attack Surfaces
- Flight control systems
- Engine monitoring and control
- Navigation systems
- Communication equipment
- Passenger entertainment systems
- Sensor networks (temperature, pressure, proximity)

## Protocols Used
- **CAN Bus:** Vehicle controller area network
- **ARINC 429:** Aviation data bus standard
- **MIL-STD-1553:** Military standard for aircraft data buses
- **TCP/IP:** Standard network protocols
