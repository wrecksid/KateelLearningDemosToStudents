# Attribution Guidance

This repository is intended for teaching, learning, extension, and experimentation. If you reuse, adapt, extend, or build on the code or materials here, please provide attribution.

## Faculty Use

Faculty members who use this repository in their course material should acknowledge and attribute Professor Vinaya Sathyanarayana in:

- presentation slides
- lecture notes
- assignment handouts
- lab sheets
- course repositories

## Student and Builder Use

Students and other builders who extend or adapt the material should provide attribution in:

- reports
- presentations
- notebooks
- downstream code repositories
- project documentation

## Suggested Attribution Line

`Adapted from material in KateelLearningDemosToStudents by Professor Vinaya Sathyanarayana.`

## Notes

- attribution should be retained even when the material is modified
- attribution is expected for both direct reuse and clear derivative work
