## Mandatory Requirements for All Users

Before using any material from this repository, please read and comply with the
**[Attribution & Usage Policy](https://github.com/VinayaSharada/KateelLearningDemosToStudents/blob/main/ATTRIBUTION.md)**.

The three non-negotiable requirements are:

1. **Credit the repo in every presentation** — include the suggested attribution line on every slide deck, handout, or published resource.
2. **Star the repository** ⭐ — [click here](https://github.com/VinayaSharada/KateelLearningDemosToStudents/stargazers)
3. **Send a usage notification email** to **vinallcontact@gmail.com** with subject `[KateelLearningDemos] Usage Notification — <Your Name / Institution>`
