# Loan Default Predictor

Predictive demo for loan default probability using machine learning.

## Learning Objectives

- Understand classification for credit risk
- Learn feature engineering for lending
- Explore model interpretability
- Understand regulatory compliance (Fair Lending)

## How to Run

1. Open `index.html` in browser
2. Enter applicant details
3. See default probability and recommendation
4. Explore feature importance

## Key Concepts Demonstrated

- **Logistic Regression**: Probability-based scoring
- **Feature Engineering**: Debt-to-income, credit history
- **Model Interpretability**: Feature importance
- **Regulatory Compliance**: Equal Credit Opportunity Act
- **Business Rules**: Minimum requirements
