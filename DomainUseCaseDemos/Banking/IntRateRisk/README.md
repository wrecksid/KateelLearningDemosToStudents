# Interest Rate Risk Management Using Synthetic Data

## Overview

This repository contains demos for Interest Rate Risk Management using synthetic financial data generated with Python Faker. It is designed for Masters level students in Finance courses focusing on applications of AI/ML in Finance.

## Files

- `generate_synthetic_data.py`: Generates synthetic Indian financial dataset (`syntheticdata.csv`) with tunable size and realistic parameters.
- `interest_rate_risk_management.py`: Demonstrates interest rate risk analysis on the synthetic data, including visualizations and sensitivity analysis.
- `syntheticdata.csv`: Sample synthetic dataset generated by `generate_synthetic_data.py`.
- `requirements.txt`: Python dependencies.
- `setup_venv.sh`: Linux bash script to set up virtual environment and install dependencies.
- `setup_venv.bat`: Windows batch script for virtual environment setup.

## Bank Use Case

Banks can use this methodology to simulate their portfolio and stress test interest rate scenarios for risk management, pricing, and capital allocation decisions. This synthetic approach enables model testing without exposing real customer data.
