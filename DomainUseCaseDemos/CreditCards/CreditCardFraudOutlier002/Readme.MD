# Banking-Outlier-CreditCardFraud

## Topic: Credit Card Fraud Detection Using AI/ML Classification

### Author: Vinaya Sathyanarayana

---

### Overview

Credit card fraud detection is an essential use case in banking, involving classification techniques to detect fraudulent transactions among valid ones.

### Domain

- **Banking**
- **Sub Domain**: Credit Cards
- **Use Case**: Detect fraud in individual credit card transactions

### Techniques Applied

- Statistical Parametric Methods
- Statistical Nonparametric Methods
- Proximity-Based Approaches
- Reconstruction-Based Approaches
- Clustering and Classification-Based Approaches

---

### Description

This project creates synthetic transaction data mimicking real-world Indian banking credit card transactions, including fraud labels. Various machine learning classification algorithms demonstrate fraud detection capabilities on this data.

---

### Self-Service Instructions for Students

- Modify parameters in `synthetic_data_generator.py` (dataset size, fraud ratio).
- Explore multiple classification algorithms in the demo script.
- Tune hyperparameters for improved fraud detection.
- Study model comparison and evaluate metrics like precision, recall.
- Visualize outputs and interpret graphs to understand decision-making.

---

### Banking Application

Banks can integrate these models in monitoring live transactions to automatically flag potentially fraudulent activity. This helps reduce financial losses and protect customers by enabling proactive fraud management.
