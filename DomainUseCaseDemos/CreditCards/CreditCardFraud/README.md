# Credit Card Fraud Detection - AI/ML Demo

This repository contains a comprehensive demonstration of machine learning techniques for credit card fraud detection, designed for a Masters-level Financial Analytics course.

## 📋 Overview

Credit card fraud detection is a critical application of machine learning in the financial sector. This demo provides:

- **Synthetic Data Generation**: Create realistic credit card transaction data with configurable fraud patterns
- **Machine Learning Models**: Multiple algorithms for fraud detection including Random Forest, Logistic Regression, and Isolation Forest
- **Business Analytics**: Comprehensive analysis of financial impact and implementation recommendations
- **Interactive Learning**: Jupyter notebooks with exercises and real-world applications
