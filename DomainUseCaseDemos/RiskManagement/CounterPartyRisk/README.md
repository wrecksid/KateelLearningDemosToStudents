# Counter Party Risk (CCPR) Assessment

Demo for evaluating counterparty credit risk in derivatives transactions.

## Learning Objectives

- Understand counterparty risk mechanics
- Learn about Credit Valuation Adjustment (CVA)
- Explore exposure modeling
- Understand netting and collateral effects

## How to Run

1. Open `index.html` in browser
2. Input counterparty details
3. See CVA calculation and exposure profile
4. Explore mitigation strategies

## Key Concepts Demonstrated

- **PD/LGD**: Probability of default and loss given default
- **Exposure Profile**: Potential future exposure
- **CVA**: Credit valuation adjustment
- **Netting**: Risk reduction through netting agreements
- **Collateral**: Margin and collateral effectiveness
