# Contagion Model

Network-based financial contagion simulation.

## Learning Objectives

- Understand contagion propagation
- Learn about systemic risk
- Explore network topology effects
- Understand regulatory implications

## How to Run

1. Open `index.html` in browser
2. Configure network parameters
3. Trigger initial shock
4. Observe contagion spread

## Key Concepts Demonstrated

- **Network Topology**: Node connectivity
- **Contagion Threshold**: Default propagation
- **Systemic Risk**: System-wide impact
- **Interconnectedness**: Correlation effects
- **Regulatory Response**: Stress testing
