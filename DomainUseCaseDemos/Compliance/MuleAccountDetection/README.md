# Mule Account Detection

Demo for detecting suspicious money laundering patterns.

## Learning Objectives

- Understand anti-money laundering (AML) techniques
- Learn behavioral anomaly detection
- Explore transaction graph analysis
- Understand regulatory compliance

## How to Run

1. Open `index.html` in browser
2. Generate synthetic transaction data
3. See flagged suspicious patterns
4. Review detection rationale

## Key Concepts Demonstrated

- **Transaction Patterns**: Velocity and frequency
- **Graph Analysis**: Connection mapping
- **Behavioral Baselines**: Anomaly scoring
- **Regulatory Reporting**: SAR filing concepts
- **False Positive Management**: Alert tuning
