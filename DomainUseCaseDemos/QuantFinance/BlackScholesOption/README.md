# Black-Scholes Option Pricing Model

Interactive demo for European option pricing using the Black-Scholes model.

## Learning Objectives

- Understand the Black-Scholes formula
- Learn about option Greeks (Delta, Gamma, Theta, Vega, Rho)
- Explore volatility impact on pricing
- Understand put-call parity

## How to Run

1. Open `index.html` in browser
2. Adjust option parameters using sliders
3. See real-time option price and Greeks
4. Explore the volatility surface

## Key Concepts Demonstrated

- **Black-Scholes Formula**: Closed-form option pricing
- **Greeks**: Risk sensitivity measures
- **Volatility**: Implied volatility input
- **Time Decay**: Theta effect
- **Put-Call Parity**: Relationship between puts and calls
