# Session 14 Assignment: Responsible AI and Fairness

## Goal

Use an existing lending or fraud demo to discuss fairness, explainability, and accountability in financial AI.

## Recommended Demos

- `DomainUseCaseDemos\CreditCards\CCUnderWriting`
- `DomainUseCaseDemos\CreditCards\CreditCardFraud`

## Student Tasks

1. Choose one decision-focused demo.
2. Identify which outputs affect customers directly.
3. Propose subgroup checks or fairness checks.
4. Explain how a business team should communicate the model to stakeholders.

## Required Outputs

- responsible-AI risk note
- fairness or explainability checklist
- recommendation memo

## Extension Ideas

- compare explainability methods
- discuss proxy variables and hidden bias
- propose a human-review fallback workflow
