# Assignments

This folder converts repository demos into course-ready assignment scaffolds.

## Intended Use

- labs
- homework
- mini projects
- mid-term case work
- final project preparation

## Suggested Assignment Pattern

Each assignment folder should include:

- learning goals
- prerequisite demo paths
- problem statement
- required outputs
- extension tasks
- evaluation rubric
