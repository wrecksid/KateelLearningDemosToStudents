## Quick Start

```bash
# Clone the repository
git clone https://github.com/VinayaSharada/KateelLearningDemosToStudents.git
cd KateelLearningDemosToStudents

# Navigate to a demo
cd TechUseCaseDemos/PatternMining/demo002

# Install and run
pip install -r requirements.txt
python main.py
```

## Learning Path

1. **Start with Pattern Mining** - Understand frequent itemsets
2. **Move to Clustering** - Group similar data points
3. **Classification** - Predict categorical outcomes
4. **Forecasting** - Predict future values
5. **Outlier Detection** - Find anomalies
