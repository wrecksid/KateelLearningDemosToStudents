## Lab Exercises

### Public Policy Resource Allocation Lab
**File**: `TechUseCaseDemos/PublicPolicyGovernance/index.html`

**Exercises**:
1. Baseline Analysis - Run default simulation
2. Education Investment - 60% education budget
3. Health Emergency - 50% healthcare budget
4. Balanced Budget - Maximize overall welfare

**Learning Outcomes**:
- Trade-off analysis in public resource allocation
- Equity metrics and social welfare optimization
- Scenario planning for policy decisions

---

### AI Governance & Compliance Lab
**File**: `TechUseCaseDemos/AIGovernancePublicSector/index.html`

**Exercises**:
1. System Classification - Select different use cases
2. Compliance Gap Analysis - Low-risk system
3. High-Risk Assessment - Healthcare triage system
4. Documentation Review - Complete assessment

**Learning Outcomes**:
- EU AI Act risk classification
- Regulatory compliance assessment
- AI governance frameworks (OECD, FTC)
