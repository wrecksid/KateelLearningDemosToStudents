# Wealth Management - Demo Catalog

Browser-based and Python demonstrations for wealth management, quantitative finance, and investment analysis.

## Course Overview

This catalog covers core wealth management concepts including portfolio theory, derivatives pricing, algorithmic trading, and behavioral finance.

| Topic | Demo | Description |
|-------|------|-------------|
| **Portfolio Theory** | [Efficient Frontier](TechUseCaseDemos/WealthManagement/efficient-frontier/) | Markowitz portfolio optimization |

## Quick Start

```bash
# Browser demos - open index.html directly
open TechUseCaseDemos/WealthManagement/efficient-frontier/index.html

# Python demos
cd TechUseCaseDemos/WealthManagement/stock-forecasting
pip install -r requirements.txt
python forecasting.py
```
